Создание окрасок

Для использования пользовательских окрасок необходимо приобрести геймпасс.

1

Скачайте шаблоны для троллейбуса, для которого планируется разработка окраски. Учтите, что для ЗиУ-9 окраска, как правило, будет совместима с ЗиУ-9 СМЕ (Система многих единиц) и, в некоторых случаях, с ЗиУ-9 Техническая помощь. Для ЗиУ-10 необходимо разработать отдельные окраски для прицепа. Для создания базовой окраски используйте любой графический редактор.После этого переходите к следующему этапу разработки.

📥 ЗиУ-9 и СМЕ📥 ЗиУ-10📥 ЗиУ-9 Техническая помощь

Templates created by symphonybreeze, apollojustice4836and oneskyvedon the forum.

2

Загрузите каждое изображение в Robloxи дождитесь полной загрузки и проверки. После этого скопируйте ID каждой текстуры, который отображается в ссылке на текстуру.

Например, ID будет выглядеть так: https://www.roblox.com/library/18926473169/YourLiveryName.

После этого переходите к следующему этапу разработки.

3

Начните настраивать вашу окраску в специальном окне ниже.Если вы хорошо разбираетесь в английском языке, вы можете выполнить настройку в TrP, перейдя в раздел Gamepass settings → Custom liveries (Click here to open livery editor)→ выберите слот (Slot_1 — Slot_15) → начните настройку. Однако мы предоставляем редактор окрасок на русском языке, который ничем не уступает редактору в TrP. Настройте каждый пункт, а затем переходите к следующему этапу разработки.

Настройка окраски

4

Скопируйте код из специального окна и вставьте его в TrP, чтобы загрузить эту ливрею. Перейдите в Gamepass settings → Custom liveries (Click here to open livery editor)→ выберите слот (Slot_1 — Slot_15) → пролистайте в самый низ → Import livery data (Paste livery data string here)→ нажмите CTRL + V. Затем дважды нажмите кнопку [← Back], а затем нажмите кнопку Save. Всё готово, ваша окраска загружена и теперь находится на аккаунте!

5

Чтобы заспавнить свою ливрею, нажмите Drive→ выберите троллейбус → Custom vehicle livery override→ выберите слот, куда вы сохранили окраску. Если хотите отключить спавн ваших окрасок в Custom vehicle livery override, выберите Disabled. Всё готово, ваша окраска заспавнена!
